# UTM Short Links Reference

> **Portfolio Tracking URLs**
>
> These short URLs redirect to your portfolio with hidden UTM tracking parameters.
> Use them when sharing your portfolio to track where visitors come from.

**Example:** `yoursite.com/li/p` → Tracks as LinkedIn Post click

## How It Works

```
1. User clicks:    yoursite.com/li/p
2. Redirects to:   yoursite.com/?utm_source=linkedin&utm_medium=post
3. GA4 records:    source=linkedin, medium=post
4. User sees:      yoursite.com/ (clean URL)
```

The user never sees the tracking parameters - they just see your clean portfolio URL.
