================================================================================
                    PORTFOLIO ANALYTICS - QUESTIONS ANSWERED
================================================================================

This document lists all the questions that can be answered using the BigQuery
SQL views and analytics pipeline built for the portfolio.

================================================================================
                         SECTION 1: VISITOR BEHAVIOR
================================================================================

1.1 Traffic & Acquisition
--------------------------
- Where are my visitors coming from? (traffic sources, referrers)
- Which traffic sources bring the most engaged visitors?
- What's my bounce rate by traffic source?
- Which campaigns are driving quality traffic?
- What countries/cities are my visitors from?
- What devices are visitors using (desktop vs mobile vs tablet)?
- What browsers are most common?
- What time of day do visitors come? (morning/afternoon/evening/night)
- What days of the week get the most traffic?
- Which landing pages perform best for each traffic source?

1.2 Session Behavior
--------------------
- How long do visitors stay on my portfolio?
- What's the average number of pages viewed per session?
- What percentage of visitors are engaged (vs bounce)?
- What's my overall engagement rate?
- How many visitors are new vs returning?
- What's the typical scroll depth across the site?
- At what scroll milestones (25%, 50%, 75%, 100%) do people stop?

1.3 Navigation Patterns
-----------------------
- How do visitors navigate through my portfolio?
- What's the most common path through sections?
- Which sections do visitors skip?
- Where do visitors enter the site? (landing pages)
- Where do visitors exit? (exit pages)
- What navigation method do visitors use? (scroll, click, menu)


================================================================================
                        SECTION 2: SECTION PERFORMANCE
================================================================================

2.1 Section Engagement
----------------------
- Which section gets the most views?
- Which section has the highest engagement rate?
- How long do visitors spend on each section?
- What's the scroll depth within each section?
- Which sections have the highest exit rate? (drop-off points)
- Which sections need improvement? (low engagement, high exit)

2.2 Section Health
------------------
- What's the health score of each section?
- Which sections are performing well vs need attention?
- Which sections should I improve first? (priority list)
- What optimization should I do for each section?
  - "improve_content" - low engagement despite high views
  - "add_cta_or_navigation" - high exit rate
  - "hook_earlier" - low scroll depth
  - "make_more_engaging" - low view duration


================================================================================
                        SECTION 3: PROJECT PERFORMANCE
================================================================================

3.1 Project Engagement
----------------------
- Which projects get the most views?
- Which projects get the most clicks?
- Which projects do people expand to read more?
- Which projects have the highest click-through rate?
- How long do visitors spend viewing each project?
- What's the average view duration per project?

3.2 Project Rankings
--------------------
- What should be the order of my projects? (data-driven ranking)
- Which projects should be featured prominently?
- Which projects are underperforming?
- How do projects rank within their category?
- What's the engagement percentile for each project?

3.3 Project Links
-----------------
- How many people click on GitHub links?
- How many people click on demo links?
- How many people click on WhatsApp/contact links?
- Which link type performs best per project?
- Which projects drive the most outbound clicks?

3.4 Case Studies
----------------
- How many visitors open case studies?
- How long do visitors spend reading case studies?
- What's the scroll depth in case studies?
- Which case studies are most engaging?


================================================================================
                       SECTION 4: CLIENT WORK & EXPERIENCE
================================================================================

4.1 Client Work Performance
---------------------------
- Which client work gets the most views?
- Which client work gets the most case study opens?
- How long do visitors spend on client case studies?
- What's the scroll depth in client case studies?
- Which clients should I highlight more?
- Which client work converts visitors best?

4.2 Domain/Industry Interest
----------------------------
- Which industries/domains attract the most interest?
  (Healthcare, Fintech, E-commerce, etc.)
- What's the ranking of domains by visitor interest?
- Which domain should I showcase more?
- Is there correlation between domain and conversion?
- Which domain attracts recruiters?

4.3 Experience/Role Interest
----------------------------
- Which job titles/roles attract the most attention?
  (Data Scientist, ML Engineer, Consultant, etc.)
- Which company names in my experience draw interest?
- What's the ranking of experiences by interest?
- Which role should I lead with on my portfolio?
- Are visitors interested in specific companies (brand value)?

4.4 Content Reading Patterns
----------------------------
- Do visitors read problem statements or skip to solutions?
- How long do visitors spend reading problem statements?
- How long do visitors spend reading solutions?
- What's the problem-to-solution reading time ratio?
- Which client's problem statements are most engaging?
- Which solutions get the most reading time?

4.5 Contribution Analysis
-------------------------
- Which specific contributions get viewed?
- What's the order of contribution views? (which ones first?)
- Are visitors reading all contributions or just the first few?


================================================================================
                         SECTION 5: SKILLS & TECHNOLOGY
================================================================================

5.1 Skill Interest
------------------
- Which skills get the most clicks?
- Which skill categories are most interesting?
- What's the ranking of skills by demand?
- Which skills should I highlight more?
- Which skills are niche vs high demand?

5.2 Technology Demand Intelligence
----------------------------------
- What technologies are visitors most interested in?
- What's the overall tech demand ranking?
- Which tech should I learn/strengthen? (learning priority)
- Which tech is high-demand vs niche?
- What tech is popular in each domain/industry?
- Is there correlation between tech interest and traffic source?
  (e.g., Do LinkedIn visitors care about different tech than Google visitors?)

5.3 Tech Stack from Client Work
-------------------------------
- Which technologies clicked from client work context?
- What tech stack is attractive per industry?
- Is Python more popular in Healthcare vs React in E-commerce?


================================================================================
                      SECTION 6: RECOMMENDATION SYSTEM
================================================================================

6.1 Recommendation Performance
------------------------------
- What's the click-through rate of recommendations?
- How many recommendations are shown vs clicked?
- What percentage of users click on recommendations?
- Is the recommendation system working?

6.2 Position Analysis
---------------------
- Which position performs best? (1st, 2nd, or 3rd)
- What's the CTR for each position?
- Should I show more or fewer recommendations?

6.3 Project-Level Recommendation Performance
--------------------------------------------
- Which projects perform best when recommended?
- Which projects have the highest recommendation CTR?
- Which source projects drive the most recommendation clicks?
- What projects should I recommend together?

6.4 System Health
-----------------
- Is the recommendation system excellent/good/needs improvement?
- What's the user conversion rate (users who click at least one rec)?


================================================================================
                          SECTION 7: CONVERSIONS
================================================================================

7.1 Contact Form Funnel
-----------------------
- How many visitors start the contact form?
- How many visitors submit the contact form?
- What's the form completion rate?
- Where do visitors drop off in the form?
- Which fields cause abandonment?

7.2 CTA Performance
-------------------
- How many CTA views vs clicks?
- What's the CTA click-through rate?
- Which CTAs perform best?
- Which CTA locations are most effective?

7.3 Resume & Downloads
----------------------
- How many resume downloads?
- What percentage of visitors download resume?
- Which traffic sources lead to resume downloads?
- What's the correlation between engagement and resume download?

7.4 Social & Outbound Links
---------------------------
- Which social platforms get the most clicks?
- What's the social click rate?
- Which outbound links are clicked most?
- What domains do visitors click to?

7.5 Publications & Certifications
---------------------------------
- Which publications get clicked?
- Which certifications get clicked?
- Which certification issuers attract interest?

7.6 Content Copy (Strong Interest Signal)
-----------------------------------------
- How often do visitors copy content?
- What type of content gets copied? (code, text, email)
- Is content copy a leading indicator of conversion?

7.7 Exit Intent
---------------
- What's the exit intent rate?
- What was the last section viewed before exit intent?
- How long were visitors on page before exit intent?
- What scroll depth before exit intent?


================================================================================
                         SECTION 8: VISITOR SEGMENTATION
================================================================================

8.1 Visitor Segments
--------------------
- What percentage are converters vs casual browsers?
- Who are my "engaged explorers"?
- Who are "high intent" visitors (resume download but no contact)?
- How many returning visitors do I have?
- What's the behavior difference between segments?

8.2 Interest Profiles
---------------------
- Who are "tech enthusiasts" (explore 5+ technologies)?
- Who are "skills focused" visitors?
- Who are "portfolio explorers" (view 4+ projects)?
- Who are "research interested" (click publications)?

8.3 Visitor Value Score
-----------------------
- What's the value score for each visitor?
- Who are my most valuable visitors?
- What behaviors indicate high-value visitors?
- Can I predict high-value visitors early in session?


================================================================================
                        SECTION 9: RECRUITER INSIGHTS
================================================================================

9.1 Recruiter Detection
-----------------------
- Which visitors are likely recruiters?
- What behaviors indicate recruiter visits?
  - Interest in experience/roles
  - Multiple domain exploration + resume download
  - Deep case study reading

9.2 Recruiter Interests
-----------------------
- What do recruiters look at most?
- Which experiences attract recruiters?
- Which domains are recruiters interested in?
- What time do recruiters typically visit?

9.3 Recruiter Conversion
------------------------
- What percentage of likely recruiters download resume?
- What percentage contact me?
- What's the recruiter-to-contact conversion rate?


================================================================================
                       SECTION 10: CONTENT QUALITY
================================================================================

10.1 Engagement Signals
-----------------------
- Which content keeps visitors engaged longest?
- What content has the highest scroll depth?
- What content gets copied? (high interest signal)
- What content leads to conversions?

10.2 Drop-off Analysis
----------------------
- Where do visitors lose interest?
- What sections have high exit rates?
- What's causing drop-offs?
- What content should I improve?

10.3 Deep Explorer Analysis
---------------------------
- Who are the "deep explorers"?
- What do deep explorers do differently?
- What content do deep explorers engage with?
- Are deep explorers more likely to convert?


================================================================================
                        SECTION 11: TREND ANALYSIS
================================================================================

11.1 Daily/Weekly Trends
------------------------
- How is traffic trending over time?
- Is engagement improving or declining?
- Are conversions increasing?
- What days perform best?

11.2 Content Performance Trends
-------------------------------
- Which projects are trending up vs down?
- Which skills are gaining interest?
- Are certain domains becoming more popular?

11.3 Recommendation System Trends
---------------------------------
- Is recommendation CTR improving?
- Are visitors engaging more with recommendations over time?


================================================================================
                      SECTION 12: ML & PREDICTIONS
================================================================================

12.1 Prediction Targets
-----------------------
- Can we predict if a visitor will convert (contact form)?
- Can we predict if a visitor will download resume?
- Can we predict engagement score?
- Can we predict if visitor is a recruiter?
- Can we predict if visitor is a "deep explorer"?

12.2 Feature Importance
-----------------------
- What features best predict conversion?
- What early session signals indicate high-value visitor?
- What behaviors correlate with recruiter visits?

12.3 Personalization
--------------------
- What projects should we recommend to each visitor?
- Can we personalize content order based on visitor behavior?
- What content should we show to different traffic sources?
